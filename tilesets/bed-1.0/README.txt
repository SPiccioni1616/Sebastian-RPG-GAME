Name:
  Beds
Version:
  1.0
Description:
  Bed tilesets created for Stendhal. The original image is created
  by bleutailfly.
Details:
  - Orientation: orthogonal
  - Dimensions: Created for 32x32 tiled maps.
  - PNG images optimized with indexed color (smaller files).
  - GIMP sources (.xcf) use RGB color.
